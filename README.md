# Explorion Waitlist

This is the waitlist landing page for **Explorion**, built with Next.js, Tailwind CSS, and shadcn/ui. 

## Getting Started

Follow these instructions to run the project locally.

### Prerequisites

Make sure you have Node.js installed on your machine (v18 or higher is recommended).

### 1. Install Dependencies

First, install all the required packages using npm:

```bash
npm install
```

### 2. Run the Development Server

Start the local development server:

```bash
npm run dev
```

## Project Structure

- `src/app/page.tsx` - The main entry point of the application.
- `src/components/ui/hero-ascii.tsx` - The core Hero component containing the Unicorn Studio animated background and the waitlist form button.
- `public/Explorion.png` - The logo used in the application.

## Technologies Used

- [Next.js](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)
- [Unicorn Studio](https://unicorn.studio/)
